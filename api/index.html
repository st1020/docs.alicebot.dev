class Bot(self, config_file = 'config.json', config_dict = None) #

Bases: object

AliceBot 机器人对象，定义了机器人的基本方法。

读取并储存配置 Config，加载适配器 Adapter 和插件 Plugin，并进行事件分发。

  • Arguments

    • config_file (Union[str, NoneType]) - 配置文件，如不指定则使用默认的 config.json。 若指定为 None，则不加载配置文件。

    • config_dict (Union[Dict, NoneType]) - 配置字典，默认为 None。 若指定字典，则会忽略 config_file 配置，不再读取配置文件。

  • Attributes

    • config (alicebot.config.MainConfig) - 机器人配置。

    • config_dict (Dict[str, Any]) - 机器人配置字典。

    • should_exit (asyncio.locks.Event) - 机器人是否应该进入准备退出状态。

    • adapters (List[alicebot.adapter.Adapter]) - 适配器列表。

    • plugins_priority_dict (Dict[int, List[Type[alicebot.plugin.Plugin]]]) - 插件优先级字典。

    • plugin_state (Dict[Type[alicebot.plugin.Plugin], Any]) - 插件状态。

    • global_state (Dict[Any, Any]) - 全局状态。

async method get(self, func = None, *, max_try_times = None, timeout = None) #

获取满足指定条件的的事件，协程会等待直到适配器接收到满足条件的事件、超过最大事件数或超时。

  • Arguments

    • func (Union[Callable[[~T_Event], Union[bool, Awaitable[bool]]], NoneType]) - 协程或者函数，函数会被自动包装为协程执行。 要求接受一个事件作为参数，返回布尔值。当协程返回 True 时返回当前事件。 当为 None 时相当于输入对于任何事件均返回真的协程，即返回适配器接收到的下一个事件。

    • max_try_times (Union[int, NoneType]) - 最大事件数。

    • timeout (Union[int, float, NoneType]) - 超时时间。

  • Returns

    Type: ~T_Event

    返回满足 func 条件的事件。

  • Raises

    • GetEventTimeout - 超过最大事件数或超时。

async method handle_event(self, current_event, *, handle_get = True, show_log = True) #

被适配器对象调用，根据优先级分发事件给所有插件，并处理插件的 stop 、 skip 等信号。

此方法不应该被用户手动调用。

  • Arguments

    • current_event (~T_Event) - 当前待处理的 Event。

    • handle_get (bool) - 当前事件是否可以被 get 方法捕获，默认为 True。

    • show_log (bool) - 是否在日志中显示，默认为 True。
